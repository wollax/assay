# T02: Add full end-to-end pipeline integration test

**Slice:** S06 — End-to-End Integration
**Milestone:** M001

## Description

No existing test chains all pipeline phases end-to-end. `test_collect_creates_target_branch` (S04) gets close but doesn't use `AssayInvoker` — it writes a custom script directly without going through the assay invocation path. This task adds `test_full_e2e_pipeline`, which exercises every phase: provision → install git → write mock assay binary → write smelt manifest → exec assay via `build_run_command` → collect result branch → teardown.

The mock assay binary is placed at `/usr/local/bin/assay` so it's on PATH, matching exactly how `AssayInvoker::build_run_command()` constructs the command (`["assay", "run", "/tmp/smelt-manifest.toml", "--timeout", "<max>"]`). The script reads the manifest, creates a file + commit in `/workspace`, and exits 0. This exercises the real `AssayInvoker` invocation path end-to-end.

Per the S06 research, this test manually chains phases rather than calling `run_with_cancellation()` — that function provisions internally, making it impossible to inject the mock assay setup step between provision and exec.

## Steps

1. Open `crates/smelt-cli/tests/docker_lifecycle.rs`. Add the following imports at the top if not already present: `smelt_core::assay::AssayInvoker`, `smelt_core::{GitCli, ResultCollector}`.

2. After `test_collect_creates_target_branch` and before the timeout tests, add `#[tokio::test] async fn test_full_e2e_pipeline()`. Begin with:
   ```rust
   let Some(provider) = docker_provider_or_skip() else { return };
   ```

3. Create a temp git repo with initial commit (inline helpers, same pattern as existing tests):
   ```rust
   let repo_dir = tempfile::tempdir().unwrap();
   let git_bin = which::which("git").expect("git on PATH");
   // run_git closure: Command::new(git_bin).args(args).current_dir(repo_dir.path()) + assert success
   run_git(&["init"]);
   run_git(&["config", "user.email", "test@example.com"]);
   run_git(&["config", "user.name", "Test"]);
   std::fs::write(repo_dir.path().join("README.md"), "# test\n").unwrap();
   run_git(&["add", "README.md"]);
   run_git(&["commit", "-m", "initial"]);
   let base_ref = /* git rev-parse HEAD */ ...;
   ```

4. Build manifest: use `test_manifest_with_repo("e2e-pipeline", repo_dir.path().to_str().unwrap())`, then set `manifest.job.base_ref = base_ref.clone()` and `manifest.merge.target = "smelt/e2e-result".to_string()`.

5. Provision container, then install git:
   ```rust
   let container = provider.provision(&manifest).await.expect("provision");
   let install = provider.exec(&container, &["sh", "-c", "apk add --no-cache git"]).await.expect("install git");
   assert_eq!(install.exit_code, 0, "git install failed: {}", install.stderr);
   ```

6. Write mock assay binary to `/usr/local/bin/assay` (base64-encoded shell script via exec):
   ```sh
   #!/bin/sh
   set -e
   cd /workspace
   git config user.email "assay@smelt.dev"
   git config user.name "Assay Mock"
   echo "generated by assay" > assay-output.txt
   git add assay-output.txt
   git commit -m "assay: generated output"
   exit 0
   ```
   Write via: `base64::engine::general_purpose::STANDARD.encode(script.as_bytes())` → exec `sh -c "echo '<b64>' | base64 -d > /usr/local/bin/assay && chmod +x /usr/local/bin/assay"`. Assert exit_code == 0.

7. Write smelt manifest, exec assay, collect, teardown, and assert:
   ```rust
   // Write manifest
   let toml = AssayInvoker::build_manifest_toml(&manifest);
   AssayInvoker::write_manifest_to_container(&provider, &container, &toml).await.expect("write manifest");

   // Exec assay via build_run_command (uses PATH lookup for "assay")
   let cmd = AssayInvoker::build_run_command(&manifest);
   let handle = provider.exec(&container, &cmd).await.expect("exec assay");
   assert_eq!(handle.exit_code, 0, "assay failed: stdout={} stderr={}", handle.stdout, handle.stderr);

   // Collect
   let git_cli = GitCli::new(git_bin.clone(), repo_dir.path().to_path_buf());
   let collector = ResultCollector::new(git_cli, repo_dir.path().to_path_buf());
   let result = collector.collect(&base_ref, "smelt/e2e-result").await.expect("collect");

   // Teardown
   provider.teardown(&container).await.expect("teardown");
   assert_container_removed(&provider, container.as_str()).await;

   // Assertions
   assert!(!result.no_changes, "should have changes");
   assert!(result.commit_count >= 1, "should have >=1 commit");
   assert!(result.files_changed.contains(&"assay-output.txt".to_string()), "expected assay-output.txt in {:?}", result.files_changed);
   // Verify branch exists on host
   let branch_check = std::process::Command::new(&git_bin)
       .args(["rev-parse", "--verify", "smelt/e2e-result"])
       .current_dir(repo_dir.path()).output().unwrap();
   assert!(branch_check.status.success(), "target branch should exist on host");
   ```
