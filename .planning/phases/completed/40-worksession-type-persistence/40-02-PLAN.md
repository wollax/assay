  **Create `crates/assay-core/src/work_session.rs`** with these public functions:

  1. **`create_work_session`**:
     ```rust
     pub fn create_work_session(
         spec_name: &str,
         worktree_path: PathBuf,
         agent_command: &str,
         agent_model: Option<&str>,
     ) -> WorkSession
     ```
     - Generate ID via `ulid::Ulid::new().to_string()`
     - Set phase to `SessionPhase::Created`
     - Include a birth transition entry: `from: Created, to: Created, timestamp: Utc::now(), trigger: "session_create"`, notes: None
     - Set `assay_version` to `env!("CARGO_PKG_VERSION")`
     - Construct `AgentInvocation` with the provided spec_name, command, and model

  2. **`transition_session`**:
     ```rust
     pub fn transition_session(
         session: &mut WorkSession,
         next: SessionPhase,
         trigger: &str,
         notes: Option<&str>,
     ) -> crate::Result<()>
     ```
     - Check `session.phase.is_terminal()` — return `WorkSessionTransition` error if true
     - Check `session.phase.can_transition_to(next)` — return `WorkSessionTransition` error if false
     - Push a new `PhaseTransition` with from=session.phase, to=next, timestamp=Utc::now(), trigger, notes
     - Update `session.phase = next`

  3. **`save_session`**:
     ```rust
     pub fn save_session(assay_dir: &Path, session: &WorkSession) -> crate::Result<PathBuf>
     ```
     - Create `.assay/sessions/` directory if needed
     - Validate session ID with `crate::history::validate_path_component(&session.id, "session ID")`
     - Serialize to pretty JSON
     - Write atomically via `NamedTempFile::new_in()` + `write_all()` + `sync_all()` + `persist()`
     - Return the final path: `sessions_dir.join(format!("{}.json", session.id))`
     - Map errors using `AssayError::io()` and `AssayError::json()` helpers

  4. **`load_session`**:
     ```rust
     pub fn load_session(assay_dir: &Path, session_id: &str) -> crate::Result<WorkSession>
     ```
     - Construct path: `assay_dir.join("sessions").join(format!("{session_id}.json"))`
     - Read file — if not found, return `WorkSessionNotFound { session_id }`
     - Deserialize JSON — map errors with `AssayError::json()`

  5. **`list_sessions`**:
     ```rust
     pub fn list_sessions(assay_dir: &Path) -> crate::Result<Vec<String>>
     ```
     - If sessions dir doesn't exist, return empty vec (not an error)
     - Read directory, filter `.json` files, extract file stems as session IDs
     - Sort the IDs lexicographically (ULID sort = chronological order)
     - Return sorted vec
