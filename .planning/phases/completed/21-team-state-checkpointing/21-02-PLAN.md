Add a `Checkpoint` variant to the `Command` enum:

```rust
/// Team state checkpointing
#[command(after_long_help = "\
Examples:
  Take a snapshot of current team state:
    assay checkpoint save

  Take a snapshot with a custom trigger label:
    assay checkpoint save --trigger \"pre-deploy\"

  Show the latest checkpoint:
    assay checkpoint show

  Show latest as JSON:
    assay checkpoint show --json

  List recent checkpoints:
    assay checkpoint list

  List more checkpoints:
    assay checkpoint list --limit 25")]
Checkpoint {
    #[command(subcommand)]
    command: CheckpointCommand,
},
```

Add `CheckpointCommand` enum:

```rust
#[derive(Subcommand)]
enum CheckpointCommand {
    /// Take a team state snapshot
    Save {
        /// Trigger label (e.g., "manual", "pre-deploy")
        #[arg(long, default_value = "manual")]
        trigger: String,
        /// Session ID to checkpoint (default: most recent)
        #[arg(long)]
        session: Option<String>,
        /// Output as JSON instead of summary
        #[arg(long)]
        json: bool,
    },
    /// Show the latest checkpoint
    Show {
        /// Output as JSON (frontmatter data only)
        #[arg(long)]
        json: bool,
    },
    /// List archived checkpoints
    List {
        /// Maximum entries to show
        #[arg(long, default_value = "10")]
        limit: usize,
    },
}
```

Add the match arm in `run()` for `Command::Checkpoint { command }`:

**Save:**
1. Get current dir as project_dir
2. Call `assay_core::checkpoint::extract_team_state(&project_dir, session.as_deref(), &trigger)`
3. Call `assay_core::checkpoint::save_checkpoint(&assay_dir(&project_dir), &checkpoint)`
4. If `--json`: print checkpoint as JSON via `serde_json::to_string_pretty`
5. Else: print summary — "Checkpoint saved: {archive_path}" + agent count + task count + trigger

**Show:**
1. Call `assay_core::checkpoint::load_latest_checkpoint(&assay_dir(&project_dir))`
2. If `--json`: print checkpoint as JSON
3. Else: read and print the raw `latest.md` file content (the human-readable markdown)

**List:**
1. Call `assay_core::checkpoint::list_checkpoints(&assay_dir(&project_dir), limit)`
2. Print a table with columns: Timestamp, Trigger, Agents, Tasks
3. If no checkpoints found, print "No checkpoints found." and exit 0

Handle errors with anyhow context:
- Missing `.assay/` → "No Assay project found. Run `assay init` first."
- Missing checkpoints dir → "No checkpoints found. Run `assay checkpoint save` to create one."
- Session not found → propagate the underlying AssayError message
