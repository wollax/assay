1. **Replace the constant:**
   - Remove: `const MAX_OUTPUT_BYTES: usize = 65_536;`
   - Add: `const STREAM_BUDGET: usize = 32_768;` with doc comment `/// Byte budget per output stream (stdout/stderr) for head+tail truncation (32 KiB).`

2. **Update `evaluate_command`** to use `truncate_head_tail` instead of `truncate_output`:

   Replace these lines (around line 549-558):
   ```rust
   let (stdout_str, stdout_truncated) = truncate_output(&stdout_raw, MAX_OUTPUT_BYTES);
   let (stderr_str, stderr_truncated) = truncate_output(&stderr_raw, MAX_OUTPUT_BYTES);

   let truncated = stdout_truncated || stderr_truncated;
   let original_bytes = if truncated {
       Some((original_stdout_len + original_stderr_len) as u64)
   } else {
       None
   };
   ```

   With:
   ```rust
   let stdout_result = truncate_head_tail(&stdout_raw, STREAM_BUDGET);
   let stderr_result = truncate_head_tail(&stderr_raw, STREAM_BUDGET);

   let truncated = stdout_result.truncated || stderr_result.truncated;
   let original_bytes = if truncated {
       Some((stdout_result.original_bytes + stderr_result.original_bytes) as u64)
   } else {
       None
   };
   ```

   Also update the references from `stdout_str`/`stderr_str` to `stdout_result.output`/`stderr_result.output` in both the `Some(exit_status)` and `None` (timeout) match arms.

3. **Remove the old `truncate_output` function** entirely (the `fn truncate_output(output: &str, max_bytes: usize) -> (String, bool)` function).

4. **Update existing tests:**
   - `truncate_output_within_budget` — rename to use `truncate_head_tail`, update assertions to use `TruncationResult` fields
   - `truncate_output_over_budget` — rename, update to check for `[truncated: ` marker format instead of `[truncated, showing last`
   - Any test referencing `MAX_OUTPUT_BYTES` — update to `STREAM_BUDGET`

5. **Add integration test** for independent stream truncation (GATE-04):

   ```rust
   #[test]

       // Use a command that produces large stdout but small stderr
       // Verify stdout is truncated, stderr is not

   }
   ```
