**app.rs — Add private helper method on App:**

Add a `resolved_specs_dir` method on `App` (impl block, private):
```rust
fn resolved_specs_dir(&self, assay_dir: &std::path::Path) -> std::path::PathBuf {
    let rel = self
        .config
        .as_ref()
        .map(|c| c.specs_dir.as_str())
        .unwrap_or("specs/");
    assay_dir.join(rel)
}
```

Per user decision: when `self.config` is `None`, fall back to `"specs/"` silently. Use `"specs/"` (with trailing slash) to match `default_specs_dir()` return value.
