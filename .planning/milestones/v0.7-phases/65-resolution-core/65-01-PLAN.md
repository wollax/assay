    - load_library_by_slug on missing slug returns Err(LibraryNotFound) with fuzzy suggestion

  <action>
Add to `crates/assay-core/src/spec/compose.rs` (below validate_slug, above the tests module):

1. `pub fn load_library(path: &Path) -> Result<CriteriaLibrary>`:
   - Mirror `load_gates` pattern exactly: `fs::read_to_string` → `toml::from_str` with `AssayError::LibraryParse` using `crate::config::format_toml_error`.
   - No validation step (unlike load_gates) — CriteriaLibrary's deny_unknown_fields handles schema validation at parse time.

2. `pub fn save_library(assay_dir: &Path, lib: &CriteriaLibrary) -> Result<PathBuf>`:
   - Call `validate_slug(&lib.name)?` first (fail before any I/O).
   - `create_dir_all(assay_dir.join("criteria"))`.
   - Use `toml::to_string_pretty(lib)` (human-authored files should be pretty-printed).
   - Atomic write: `NamedTempFile::new_in(&criteria_dir)` → `write_all` → `as_file().sync_all()` → `persist(&final_path).map_err(|e| AssayError::io(..., e.error))`.
   - Return final path.
   - Import `tempfile::NamedTempFile` and `std::io::Write`.

3. `pub fn scan_libraries(assay_dir: &Path) -> Result<Vec<CriteriaLibrary>>`:
   - `let criteria_dir = assay_dir.join("criteria");`
   - If `!criteria_dir.is_dir()` return `Ok(vec![])`.
   - `read_dir(&criteria_dir)` → filter `.toml` extension → load each with `load_library` → collect successes, skip errors (log nothing — consistent with scan() for now).
   - Sort by `lib.name`.
   - Return the sorted vec.

4. `pub fn load_library_by_slug(assay_dir: &Path, slug: &str) -> Result<CriteriaLibrary>`:
   - Call `validate_slug(slug)?`.
   - Build path: `assay_dir.join("criteria").join(format!("{slug}.toml"))`.
   - If file doesn't exist: scan for available slugs, call `find_fuzzy_match(slug, &available)` from `crate::spec`, return `AssayError::LibraryNotFound { slug, criteria_dir, suggestion }`.
   - Note: `find_fuzzy_match` is `pub(crate)` so accessible from within assay-core.
   - If file exists: call `load_library(&path)`.

5. Add tests to the `#[cfg(test)] mod tests` block using `tempfile::tempdir()` for filesystem operations.
