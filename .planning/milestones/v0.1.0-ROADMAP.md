## Milestone Summary

**Key Decisions:**
- assay-types = pub DTOs, zero logic; assay-core = free functions, all behavior
- TOML for config, JSON for data exchange
- Enum dispatch for gates, not trait objects
- Gate is pure config, not mixed config+state (GateResult is runtime state)
- assay-mcp as library crate; single `assay` binary for all surfaces
- spawn() + reader threads + try_wait polling for gate timeout support
- Skills use MCP tool orchestration, not shell commands
- MCP Spike: GO — rmcp 0.17 + stdio confirmed

**Issues Deferred:**
- No CLI integration tests (assay-cli has 0 automated tests)
- Missing VERIFICATION.md for phases 8-10
- Agent-evaluated criteria track (deferred to v0.2+)

**Technical Debt Incurred:**
- duration_ms > 0 not asserted in evaluate_echo_hello test
- Skipped criteria not named in streaming output (count only)
- 30 open issues from PR reviews (9 from Phase 8 review)
